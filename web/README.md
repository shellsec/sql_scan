# WEB 控制台支持
## 启动服务
```
cd ./web/
python app.py
```
## 使用浏览器打开
http://ipaddress:5000/
## 搜索功能
http://ipaddress:5000/q?s=host:baidu.com
```
host:baidu.com
port:80,443
status_code:200
method:POST
scheme:https
extension:php
content_type:text/html
```
